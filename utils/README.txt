This directory contains useful scripts and tools provided by GGG users:

ncep_download:

    - bash-based automatic NCEP data download script for Linux/Unix
    - maintained by Dietrich Feist <dfeist@bgc-jena.mpg.de>

python:

    - Python utilities to work with TCCON data
    - maintained by Dietrich Feist <dfeist@bgc-jena.mpg.de>
